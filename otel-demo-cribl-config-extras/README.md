Want a friendly herd to practice on? Start with the [OpenTelemetry Demo docs](https://opentelemetry.io/docs/demo/), which show you how to spin up the demo and try a few “see-OTel-in-action” scenarios.

Here in this directory, you’ll find an example `otelcol-config-extras.yml` you can use in the demo's [docker deployment](https://opentelemetry.io/docs/demo/docker-deployment/#bring-your-own-backend) to send OTLP logs, metrics, and traces to Cribl.

If you’d like to see the original, un-goatified version, check out the demo’s GitHub repo and the unedited `otelcol-config-extras.yml` [here](https://github.com/open-telemetry/opentelemetry-demo/blob/main/src/otel-collector/otelcol-config-extras.yml).
